Installation

It’s very easy and simple to implement this navigation menu on your website. You can use the live style generator, where you can set more than 20 options to create with no effort your custom menu and then click on “Get The Generated Code” button and follow the steps.

Make sure you include all the necessary files and folders, and the paths to the files are right.

If you don't want to use the generator, take a look at the examples code, and bellow snippets.

Menu HTML Markup

Basic HTML Markup

To create your meganizr menu you only need to create an unordered list (<ul>) with the required class .meganizr. If you want to add more functionality to the menu you can use the following classes.

Class Description
.modernizr Load modernizr styles
.mzr-slide For slide transitions
.mzr-fade For fade in/out transitions
.mzr-responsive For responsive menu 
<ul class="meganizr">
    <!-- Menu Items Here -->
</ul>

Link with Home Icon

To add a link with home icon instead of text use the class .mzr-home-dark for a dark icon and .mzr-home-light for a light icon.

Class Description
.mzr-home-dark Add the dark home icon
.mzr-home-light Add the light home icon 
<li class="mzr-home-dark"><a href="#">Home</a></li>

Normal Link (no dropdown)

To add a normal link, no dropdowns, just add a new list item (<li>) with an anchor tag (<a>) inside.

<li><a href="#">Normal Link</a></li>

Link with Mega Dropdown

To add a mega dropdown link, add a list item (<li>) with the class .mzr-drop then add an anchor tag (<a>) and a <div> inside. To use the content preset styles add the class .mzr-content and to set the mega dropdown size use one of the following classes (ex: .drop-four-columns).

Class Description
.mzr-drop Required to make the dropdowns work.
.mzr-content Load preset content styles
.drop-six-columns Create a dropdown box that allows up to six columns in a row
.drop-five-columns Create a dropdown box that allows up to five columns in a row
.drop-four-columns Create a dropdown box that allows up to four columns in a row
.drop-three-columns Create a dropdown box that allows up to three columns in a row
.drop-two-columns Create a dropdown box that allows up to two columns in a row 
<li class="mzr-drop">

    <a href="#">6 Columns</a>

    <div class="mzr-content drop-six-columns">
        Content here ...
    </div>

</li>

Link with Dropdown Levels

To add a link with dropdown levels create an unordered list (<ul>) for each dropdown level. You can align them to the right (default) or left (.mzr-align-left) and use as many dropdown levels as you want, there’s no limit.

Class Description
.mzr-drop Required in all <li> that have a new dropdown level.
.mzr-levels Required on the first <li>.
.mzr-align-left Align the dropdown level on the left of his parent. 
<li class="mzr-drop mzr-levels">
    <a href="#">Dropdown</a>
    <ul>
        <li><a href="#">First Level</a></li>
        <li class="mzr-drop">
            <a href="#">First Level</a>
            
            <ul>
                <li><a href="#">Second Level</a></li>
                <li><a href="#">Second Level</a></li>
                <li><a href="#">Second Level</a></li>
                <li class="mzr-drop mzr-align-left">
                    <a href="#">Second Level</a>
                   
                    <ul>
                        <li><a href="#">Third Level</a></li>
                        <li><a href="#">Third Level</a></li>
                        <li><a href="#">Third Level</a></li>
                        <li><a href="#">Third Level</a></li>
                        <li><a href="#">Third Level</a></li>
                    </ul>
               
                </li>
                <li><a href="#">Second Level</a></li>
            </ul>
        
        </li>
        <li><a href="#">First Level</a></li>
        <li><a href="#">First Level</a></li>
        <li><a href="#">First Level</a></li>
    </ul>
</li>

Align Links on the Right

To position a link to the right, add the class .mzr-align-right to the <li>.

Class Description
.mzr-align-right Align The link to the right. 
<li class="mzr-align-right"><a href="#">Right Link</a></li>

Contact Form

This menu have available a styled contact form. To use it copy the following code and paste it inside of <ul class="meganizr">.

Class Description
.mzr-drop Required for dropdown to work.
.mzr-contact Required if you're using the contact form. 
<li class="mzr-drop mzr-contact">

    <a href="#">Contact</a>

    <div class="mzr-content drop-two-columns">

        <div class="two-col">
            <form action="#" method="#" id="mzr-contact-form">
                <input type="text" name="name" id="name" placeholder="Name">
                <input type="email" name="email" id="email" placeholder="email@email.com">
                <textarea name="message" id="message" placeholder="Message"></textarea>
                <input type="submit" value="Send Message">
            </form>
        </div>

    </div>

</li>

Search Form

To add the search field to the menu, copy the following code.

Class / Id Description

#search-form Required id to slyle the form input.

        <input type="text" id="search-form" name="search-form" value="Search" onfocus="if(this.value=='Search')this.value='';" onblur="if(this.value=='')this.value='Search';">
        <input type="submit" value="">
    </form>

</li>

CSS Files and Structure

This menu uses two CSS files. The meganizr.css file contains all the styles and functionality of the menu and the meganizer-ie.css contains some fixes for internet explorer 8 and 7. The meganizr.css files is dived by topics, if you want to edit it, scroll down to the topic you’re looking for and find the CSS rule that contains that style.

File Description
meganizr.css Contains all the Meganizr styles.
meganizr-ie.css Add support for IE8 and IE7. 
/* ###############################################

    // MEGANIZR.CSS FILE STRUCTURE

    // 1. RESET STYLES
    // 2. NAVIGATION BAR STYLES
    // 3. SEARCH FORM
    // 4. DROPDOWN
    // 5. CONTENT STYLES
    // 6. RESPONSIVE
    // 7. EXTRA COLOR STYLES

############################################### */
