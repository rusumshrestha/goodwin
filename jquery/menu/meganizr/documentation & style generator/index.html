Features

Fully Customizable


  • This menu was created in pure CSS3, which makes you able to change every bit of it.
  • Easily create your custom style by using the Live Style Generator.
  • Supports Slide and Fade transitions. You can also disable theme completely by removing a single class.

Responsive Layout


  • This is a responsive menu that uses media queries to adapt to the user screen resolution.
  • You don’t need to worry anymore about your users screen resolution as this menu will work with any resolution.

Mega Dropdowns


  • Full support for mega menus (up to six columns) and unlimited dropdown levels with left and right alignment.
  • Built using a columns layout which makes it easy to organize the content of mega menus.
  • You can add any type of content in any language (HTML, PHP, JS, etc.).

Styled Contact & Search Form


  • This menu has a full styled contact form and search form.
  • Just add your custom contact and search script to start using them.

Cross Browser Support


  • This menu was tested and it’s working in all major browsers, including the old Internet Explorer 7.
  • Some CSS3 features are not supported in older browsers, the gradients will fallback to a solid color and the rounded corners and shadows will be removed.

Free Updates & Support


  • If you buy this menu you will get free support and free life time updates to new features and bugs fixes.
  • For support questions open a new ticket, I will be happy to help you :).
